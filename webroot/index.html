Endpoints

GET://api/v1/aggregations

Lists all aggregations currently in use by the aggregator

GET://api/v1/aggregations/{name}

Lists all aggregations who's name or key contains {name}

GET://api/v1/aggregates

Lists all aggregates currently available

GET://api/v1/aggregates/{name}

Lists all aggregates who's name or key contains {name}

GET://api/v1/aggregate/{id}

Returns the aggreate for {id}
